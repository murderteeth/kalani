# Zookeeper
Yearn vault automator

## lfg
```sh
cp packages/web/.env.example packages/web/.env
# configure packages/web/.env
bun install
bun dev
```
Head to [http://localhost:3000](http://localhost:3000)

## production
```sh
bun run build
```
